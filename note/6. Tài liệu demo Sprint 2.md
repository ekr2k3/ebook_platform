# 🔧 5. Chi tiết User Story (technical)

---

## 🟢 US-B04 – Mua ebook

### Acceptance Criteria

- Given buyer chọn ebook
    
- When nhấn “Mua”
    
- Then hệ thống tạo order trạng thái PENDING
    

### Task

- tạo API `/orders/create`
    
- lưu order + order_items
    
- redirect sang payment
    

---

## 🟢 US-B05 – Cấp quyền ebook

### Acceptance Criteria

- Given thanh toán thành công
    
- When callback từ payment
    
- Then hệ thống thêm ebook vào `user_ebooks`
    

### Task

- xử lý webhook payment
    
- update order status = SUCCESS
    
- insert `user_ebooks`
    

---

## 🟢 US-B07 – Thư viện cá nhân

### Acceptance Criteria

- Given user đăng nhập
    
- When vào thư viện
    
- Then hiển thị ebook đã mua
    

### Task

- query `user_ebooks`
    
- join với `ebooks`
    
- render Thymeleaf page
    

---

## 🟢 US-B06 – Reader online

### Acceptance Criteria

- Given user sở hữu ebook
    
- When mở ebook
    
- Then hiển thị nội dung
    

### Task

- kiểm tra quyền truy cập
    
- load file từ storage
    
- dùng PDF viewer (iframe/pdf.js)
    

---

## 🔴 US-A03 – Admin quản lý payment

### Acceptance Criteria

- Given admin truy cập
    
- When xem danh sách order
    
- Then hiển thị trạng thái giao dịch
    

### Task

- admin page `/admin/orders`
    
- filter theo status
    
- pagination

# 🧪 6. Definition of Done (DoD)

### Functional

- Mua ebook thành công end-to-end
    
- Ebook xuất hiện trong thư viện
    
- Reader mở được ebook
    

### Data consistency

- Không tạo duplicate order
    
- Order và quyền truy cập đồng bộ
    

### Security

- Chỉ user mua mới đọc được ebook
    
- Không truy cập trái phép file
    

### Payment

- Callback xử lý đúng
    
- Không double charge
    

### Code

- Pass review
    
- Không lỗi runtime

# ⚠️ 7. Rủi ro

---

## 💳 Payment lỗi / double charge

**Rủi ro**

- user bị trừ tiền 2 lần
    
- webhook gọi nhiều lần
    

**Giải pháp**

- idempotency key
    
- check order status trước update
    

---

## 🔐 Access control sai

**Rủi ro**

- user chưa mua vẫn đọc được
    

**Giải pháp**

- check `user_ebooks` trước khi render
    

---

## 📦 File leak

**Rủi ro**

- truy cập trực tiếp file URL
    

**Giải pháp**

- không expose file public
    
- stream file qua backend
    

---

## ⚡ SSR performance

**Rủi ro**

- reader load chậm
    

**Giải pháp**

- lazy load
    
- pagination library
