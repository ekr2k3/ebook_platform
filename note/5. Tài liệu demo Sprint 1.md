# 🎯 1. Sprint Goal

> Xây dựng nền tảng hệ thống ebook với các chức năng cốt lõi:

- Người dùng đăng ký, đăng nhập và phân quyền
    
- Seller có thể upload ebook và nhập metadata
    
- Admin có thể duyệt ebook
    
- Buyer có thể tìm kiếm và xem chi tiết ebook đã được duyệt
    

👉 Mục tiêu: tạo **vertical slice hoàn chỉnh từ nội dung → duyệt → hiển thị**

# ✅ 4. Definition of Done (DoD)

Một user story được xem là hoàn thành khi:

### 🔧 Functional

- Chức năng hoạt động đúng theo acceptance criteria
    
- Không có lỗi critical/blocker
    

### 🔐 Security

- Password được hash (BCrypt)
    
- Phân quyền hoạt động đúng
    
- Không truy cập trái phép (admin/seller)
    

### 💻 Code

- Code đã review
    
- Không có lỗi compile
    
- Follow coding convention
    

### 🧪 Testing

- Test manual đầy đủ flow
    
- API hoạt động đúng
    

### 🚀 Deploy

- Build thành công
    
- Deploy lên môi trường test chạy được
